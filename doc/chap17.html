17.1 Constructors of Premaniplexes

17.1-1 Premaniplex
‣ Premaniplex( group )( operation )

Returns: IsPremaniplex.

Given a group we return the premaniplex with that group as its connection group. Note: We will assume (but not check) that the group is an sggi.

Here we build a premaniplex with 3 flags.

gap> g:=Group((1,2),(2,3),(1,2));;
gap> Premaniplex(g);
Premaniplex of rank 3 with 3 flags

17.1-2 Premaniplex
‣ Premaniplex( edgelabeledgraph )( operation )

Returns: IsPremaniplex.

Given an edge labeled graph we return the premaniplex with for that graph. Note: We will assume (but not check) that the graph is a premaniplex, that is to say, we are assumging each vertex is incident with one edge of each label.

Here we have a premaniplex with 2 flags.

gap> gap> L:=[[[1,2],0], [[1,2],1], [[1],2], [[2],2]];;
gap> F:=EdgeLabeledGraphFromLabeledEdges(L);;
gap> Premaniplex(F);
Premaniplex of rank 3 with 2 flags
