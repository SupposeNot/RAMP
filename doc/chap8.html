8.1-1 UniversalSggi

In the first form, returns the universal Coxeter Group of rank n. In the second form, returns the Coxeter Group with Schlafli symbol sym.

8.1-2 ReflexibleManiplex
‣ ReflexibleManiplex( g )( operation )

Given a group g (which should be a string C-group), returns the abstract regular polytope with that automorphism group, where the privileged generators are those returned by GeneratorsOfGroup(g).

8.1-3 ReflexibleManiplex
‣ ReflexibleManiplex( symbol, relations )( operation )

Returns an abstract regular polytope with the given Schlafli symbol and with the given relations. The formatting of the relations is quite flexible. All of the following work:

q := ReflexibleManiplex([4,3,4], "(r0 r1 r2)^3, (r1 r2 r3)^3");
q := ReflexibleManiplex([4,3,4], "(r0 r1 r2)^3 = (r1 r2 r3)^3 = 1");
p := ReflexibleManiplex([infinity], "r0 r1 r0 = r1 r0 r1");

If the option set_schlafli is set, then we set the Schlafli symbol to the one given. This may not be the correct Schlafli symbol, since the relations may cause a collapse, so this should only be used if you know that the Schlafli symbol is correct.

8.1-4 ReflexibleManiplex
‣ ReflexibleManiplex( name )( operation )

Returns the regular polytope with the given symbolic name. Examples: ReflexibleManiplex("{3,3,3}"); ReflexibleManiplex("{4,3}_3"); If the option set_schlafli is set, then we set the Schlafli symbol to the one given. This may not be the correct Schlafli symbol, since the relations may cause a collapse, so this should only be used if you know that the Schlafli symbol is correct.

8.1-5 Maniplex
‣ Maniplex( G )( operation )

Returns a maniplex with connection group G, where G is assumed to be a permutation group on the flags.
