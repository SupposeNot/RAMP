16.1 Voltage Operator

16.1-1 VoltageOperator
‣ VoltageOperator( etain, etaout, Xa )( operation )

Returns: IsManiplex

Returns the output of the voltage operator acting on Xa. Xa is a n-premaniplex as an edge labeled graph, Y is a m-premaniplex. eta is a voltage assignment on the darts of Y. etain is a list of all darts of Y. etaout is a string giving words in the universal sggi of rank n. If Xa is given as a maniplex, the operation is done to its flag graph.
