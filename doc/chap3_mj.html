3.2 Schlafli symbol

3.2-1 SchlafliSymbol
‣ SchlafliSymbol( M )( attribute )

Returns the Schlafli symbol of the maniplex M. Each entry is either an integer or a set of integers, where entry number i shows the polygons that we obtain as sections of (i+1)-faces over (i-2)-faces.

3.2-2 IsEquivelar
‣ IsEquivelar( M )( property )

Returns: the the maniplex M is equivelar; i.e., whether its Schlafli Symbol consists of integers at each position (no lists).

3.3-3 IsTight
‣ IsTight( P )( property )

Returns: true or false

Returns whether the polytope P is tight, meaning that it has a Schlafli symbol {k_1, ..., k_{n-1}} and has 2 k_1 ... k_{n-1} flags, which is the minimum possible. This property doesn't make any sense for non-polytopal maniplexes, which aren't constrained by this lower bound.

3.3-4 IsDegenerate
‣ IsDegenerate( M )( property )

Returns: true or false

Returns whether the maniplex M has any sections that are digons. We may eventually want to include maniplexes with even smaller sections.

3.4 Zigzags and holes

3.4-1 ZigzagLength
‣ ZigzagLength( M, j )( operation )

Returns: The lengths of j-zigzags of the 3-maniplex M. This corresponds to the lengths of orbits under r0 (r1 r2)^j.

3.4-2 ZigzagVector
‣ ZigzagVector( M )( attribute )

Returns: The lengths of all zigzags of the 3-maniplex M. A rank 3 maniplex of type {p, q} has Floor(q/2) distinct zigzag lengths because the j-zigzags are the same as the (q-j)-zigzags.

3.4-3 PetrieLength

Returns: The length of the petrie polygons of the maniplex M.
