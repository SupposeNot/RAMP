4.1 Constructors

4.1-1 UniversalSggi
‣ UniversalSggi( n )( operation )
‣ UniversalSggi( sym )( operation )

In the first form, returns the universal Coxeter Group of rank n. In the second form, returns the Coxeter Group with Schlafli symbol sym.

4.1-2 AbstractRegularPolytope
‣ AbstractRegularPolytope( g )( operation )

Given a group g (which should be a string C-group), returns the abstract regular polytope with that automorphism group, where the privileged generators are those returned by GeneratorsOfGroup(g).

4.1-3 AbstractRegularPolytope
‣ AbstractRegularPolytope( symbol, relations )( operation )

Returns an abstract regular polytope with the given Schlafli symbol and with the given relations. The formatting of the relations is quite flexible. All of the following work:

q := AbstractRegularPolytope([4,3,4], "(r0 r1 r2)^3, (r1 r2 r3)^3");
q := AbstractRegularPolytope([4,3,4], "(r0 r1 r2)^3 = (r1 r2 r3)^3 = 1");
p := AbstractRegularPolytope([infinity], "r0 r1 r0 = r1 r0 r1");

4.1-4 AbstractRegularPolytope
‣ AbstractRegularPolytope( name )( operation )

Returns the regular polytope with the given symbolic name. Examples: AbstractRegularPolytope("{3,3,3}"); AbstractRegularPolytope("{4,3}_3");
