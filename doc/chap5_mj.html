5.1 Regular polyhedra

5.1-1 DegeneratePolyhedra
‣ DegeneratePolyhedra( maxsize )( function )

Returns all degenerate polyhedra (of type {2, q} and {p, 2}) with up to maxsize flags.

5.1-2 FlatRegularPolyhedra
‣ FlatRegularPolyhedra( maxsize )( function )

Returns all nondegenerate flat regular polyhedra with up to maxsize flags. Currently supports a maxsize of 4000 or less.

5.1-3 SmallRegularPolyhedra
‣ SmallRegularPolyhedra( sizerange )( function )

Returns all regular polyhedra with sizes in sizerange flags. Currently supports a maxsize of 4000 or less. You can also set options "nondegenerate" and "nonflat".

L1 := SmallRegularPolyhedra(500);;
L2 := SmallRegularPolyhedra(1000 : nondegenerate);;
L3 := SmallRegularPolyhedra(2000 : nondegenerate, nonflat);;
